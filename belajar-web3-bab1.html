Bab 1: Evolusi Internet (Web 1.0, 2.0, 3.0)

Memahami dari mana kita berasal dan ke mana kita akan pergi.

Web 1.0: "Read-Only" (Si Statis)

Bayangkan internet di tahun 1990-an. Halaman web statis seperti brosur digital. Anda hanya bisa **membaca** informasi. Tidak ada login, tidak ada komentar, tidak ada media sosial. Interaksi Anda terbatas pada mengklik link. Ini adalah internet "satu arah".

Web 2.0: "Read-Write" (Si Sosial)

Ini adalah internet yang kita gunakan sekarang (Facebook, Google, Twitter, TikTok). Anda tidak hanya membaca, tapi juga **menulis** dan berinteraksi. Anda mengunggah foto, menulis status, dan memberi komentar. Masalahnya? Semua data Anda, konten Anda, dan identitas Anda **dimiliki dan dikontrol** oleh perusahaan-perusahaan besar tersebut. Anda adalah "produknya".

Web 3.0: "Read-Write-Own" (Si Desentralisasi)

Inilah ide besarnya. Web3 menggunakan teknologi blockchain untuk mengembalikan **kepemilikan** kepada pengguna.

  • Alih-alih login dengan Google, Anda login dengan "dompet" (wallet) kripto Anda.
  • Konten dan aset digital Anda (seperti NFT) adalah milik Anda, tercatat di blockchain.
  • Aplikasi (dApps) berjalan di jaringan terdesentralisasi, bukan di server satu perusahaan.

Web3 adalah tentang memindahkan kekuasaan dari platform terpusat kembali ke individu. Di bab selanjutnya, kita akan membahas salah satu pilar utamanya: NFT.
